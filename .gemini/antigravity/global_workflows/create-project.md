# Create Project

Your task is to create folders for the project documents


# Parameters

project_name - Name of the project
release - Name of the release. Default to "latest"


# Usage

create_project --project_name "Project Name" --release release_7.5


# Output

1) Ensure you are working in the root of the github repo -(https://github.com/sandeep-i2o/i2o-project-ai-docs)
2) Create folder in projects directory with name {{project_name}}/{{release_version}}/docs
3) Create subfolders in {{project_name}}/{{release_version}}/docs directory with name design, requirements, tests
