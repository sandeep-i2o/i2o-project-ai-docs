# Create Project

Your task is to create folders for the project documents


# Parameters

project_name - Name of the project


# Usage

create_project --project_name "Project Name"


# Output

1) Ensure you are working in the root of the github repo - [i2o-aicellerate] (https://github.com/sandeep-i2o/i2o-feature-ai-docs)
2) Create folder in features directory with name {{project_name}}/docs
3) Create subfolders in {{project_name}}/docs directory with name design, requirements, tests
