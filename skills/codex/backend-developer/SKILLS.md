# Skills: Java Backend Engineer (Java 8 + Spring Boot)

Focusing on high-performance backend development within agentic environments like **[Google Antigravity](https://antigravity.google)**.

## ☕ Core Java 8 Mastery
*   **Functional Programming:** Proficiency with [Lambda Expressions](https://docs.oracle.com) and Functional Interfaces (`Predicate`, `Function`, `Consumer`).
*   **Stream API:** Declarative data processing using `.filter()`, `.map()`, and `.collect()` to replace imperative loops.
*   **Concurrency:** Asynchronous programming using [CompletableFuture](https://docs.oracle.com) for non-blocking I/O.
*   **Modern APIs:** Implementation of the thread-safe `java.time` package and `Optional<T>` for null-safety.

## 🍃 Spring Boot Ecosystem
*   **Rapid Development:** Utilization of [Spring Initializr](https://start.spring.io) and Starters for automated configuration.
*   **Data Access:** Persistence layer management using [Spring Data JPA](https://spring.io) and Hibernate.
*   **Security:** Securing RESTful endpoints with [Spring Security](https://spring.io) (JWT, OAuth2).
*   **Observability:** Monitoring application health and metrics via [Spring Boot Actuator](https://docs.spring.io).

## 🚀 Agentic Development (Antigravity)
*   **Agent Prompting:** Writing precise technical instructions for [Antigravity Agents](https://antigravity.google) to refactor legacy code or generate boilerplate.
*   **Workflow Automation:** Leveraging the **Agent Manager** to parallelize unit testing and API documentation.
*   **Integration Testing:** Using the [Antigravity Browser](https://antigravity.google) for automated end-to-end testing of Spring Boot controllers.

## 🛠️ Infrastructure & Tools
*   **Build Tools:** Lifecycle management and dependency resolution using **Maven** or **Gradle**.
*   **Containerization:** Packaging microservices using **Docker** for consistent deployment.
*   **Version Control:** Advanced [Git](https://github.com) workflows (branching, merging, and pull request reviews).
